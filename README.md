#  echo-client-server гайд
Простая программа для локальной сети чтобы подключиться к другому серверу

## **Для сервера необходимо:**
1)Узнать свой **IP**

2)вбить его в echo-server.py в строчку HOST

3)Запустить

## **Для клиента необходимо:**
1)узнать **IP** сервера

2)вбить его в **echo-client.py** в строчку HOST

3)Запустить

# **Чтобы узнать, что сработало - в терминале должно выдать Hello world**
